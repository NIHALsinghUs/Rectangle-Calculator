Rectangle Calculator

A beginner-friendly Python project that calculates the area and perimeter of a rectangle using Object-Oriented Programming (OOP).

Features
Takes user input for length and width
Calculates area of rectangle
Calculates perimeter of rectangle
Uses Python classes and methods
Demonstrates use of __init__() and f-strings
Formula Used

Area: Area=l×w

Perimeter: Perimeter=2(l+w)

Technologies Used : Python 3

Example Output
Enter the length : 10
Enter the width : 5

Length : 10, Width : 5
Area of rectangle : 50
Perimeter of rectangle : 30
